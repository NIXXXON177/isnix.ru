Инструкция

Как начать играть на ISTHISNIXXXON

Сервер на Fabric 1.21.1. Рекомендуем только ElyPrismLauncher и готовую сборку с сайта.

Скачать сборку (~39 МБ) ElyPrismLauncher
🚀
Шаг 1

Установи ElyPrismLauncher

  • Скачай лаунчер с GitHub Releases. На Windows, если не уверен — бери ElyPrismLauncher-Windows-MSVC-Setup (последняя версия).
  • Установи и запусти. Войди через свой аккаунт Minecraft (Microsoft) или офлайн-ник — как настроишь в лаунчере.
  • Другие лаунчеры мы не рекомендуем — на сервере тестируем только связку ElyPrismLauncher + Fabric 1.21.1.
📦
Шаг 2

Скачай и установи сборку сервера

На главной в разделе «Скачать» или по кнопке выше — архив ISTHISNIXXXONmods.zip с модами сервера.

  • Вариант А (проще): в ElyPrismLauncher нажми «Добавить» → «Импортировать из ZIP» и выбери скачанный архив. Лаунчер создаст инстанс с нужной версией и модами.
  • Вариант Б: создай инстанс вручную — Minecraft 1.21.1, загрузчик Fabric. Распакуй ZIP и скопируй папку mods (и при необходимости другие файлы из архива) в папку этого инстанса.

💡 Путь к папке инстанса: правый клик по инстансу в лаунчере → «Папка инстанса» / «Открыть папку модов».

🔊
Шаг 3

Проверь моды и запусти игру

В сборке уже есть обязательный Simple Voice Chat, карта Xaero's, OPAC (приваты и кланы), FallingTree, trade и другие моды сервера. Если ставишь вручную — убедись, что в папке mods лежат jar-файлы для Fabric 1.21.1.

  • В лаунчере выбери инстанс ISTHISNIXXXON / Fabric 1.21.1 и нажми «Играть».
  • Добавить мод отдельно: в ElyPrismLauncher открой инстанс → «Моды» → Modrinth / CurseForge, версия 1.21.1, загрузчик Fabric.
🎮
Шаг 4

Заходи на сервер

  • IP: mc.isnix.ru (статус и онлайн)
  • Вайтлист — только на сайте: личный кабинет (ник, как обращаться, зачем на сервер). В соцсетях — только новости.
  • Заходи только после одобрения — иначе сервер не пустит.
🛡️
После входа

Кланы и приваты (OPAC)

  • Меню мода: клавиша ' (апостроф) → приваты и настройки.
  • Клан: /openpac-parties create → invite <ник> → друг: /openpac-parties join <ник_лидера>
  • Чат клана: /opm <текст>
  • Тег клана в TAB: ' → My player config → Party name (например [Wolves]). Цвет/жирность — /clantag color green, /clantag bold on (только владелец клана).
🤝
После входа

Торговля без валюты

  • Прямой обмен: /trade <ник> — окно обмена с игроком.
  • Рынок лотов: /sell — выставить предмет, оплата только ресурсами (без монет).
  • Чат: локально рядом; глобально — сообщение с префиксом ! в начале (см. правила на главной).

Что в сборке ISTHISNIXXXONmods.zip

Клиентские моды для игры на сервере (обновляй ZIP с главной после крупных апдейтов):

  • Обязательно: Fabric API, Simple Voice Chat, OPAC, styled-chat, Xaero's World Map, Sodium/Lithium (оптимизация FPS)
  • QoL: FallingTree, trade, doubledoors, fsit, rightclickharvest, FastRTP, ItemFrament, Female Gender Mod
  • Только на сервере (в ZIP не нужны): EasyAuth, LuckPerms, TAB, GrimAC, isnix-market (/sell), админ-моды
