Инструкция

Как начать играть на ISTHISNIXXXON

Только для ПК: Minecraft Java Edition 1.21.1, NeoForge, ElyPrismLauncher и сборка модов с сайта.

Скачать сборку (~480 МБ) ElyPrismLauncher

Кабинет и вайтлист — иногда только через VPN

Регистрация и заявка на сайте идут через api.isnix.ru. Если личный кабинет не открывается — включи Cloudflare WARP (1.1.1.1) или VPN. Подключение к серверу mc.isnix.ru — без VPN.

Шаг 1

Установи ElyPrismLauncher

  • Скачай лаунчер с GitHub Releases. На Windows, если не уверен — бери ElyPrismLauncher-Windows-MSVC-Setup (последняя версия).
  • Установи и запусти. Войди через свой аккаунт Minecraft (Microsoft) или офлайн-ник — как настроишь в лаунчере.
  • Другие лаунчеры мы не рекомендуем — на сервере тестируем только связку ElyPrismLauncher + NeoForge 1.21.1.
Шаг 2

Скачай и установи сборку сервера

На главной в разделе «Скачать» или по кнопке выше — архив ISTHISNIXXXONmods.zip — клиентские моды для игры на сервере.

Вариант А · проще

В ElyPrismLauncher нажми «Добавить» → «Импортировать из ZIP» и выбери скачанный архив. Лаунчер создаст инстанс с нужной версией и модами.

Вариант Б · вручную

Создай инстанс: Minecraft 1.21.1, загрузчик NeoForge. Распакуй ZIP и скопируй все .jar из папки mods архива в папку mods инстанса (правый клик → «Открыть папку модов»).

Путь к папке инстанса: правый клик по инстансу в лаунчере → «Папка инстанса» / «Открыть папку модов».

Шаг 3

Проверь моды и запусти игру

В сборке — только клиентские моды: голосовой чат, Create, Sable, Irons Spells, OPAC, карты Xaero's, Sodium, JEI. FallingTree, /sell и FastRTP — на сервере, отдельно не ставь. Если собираешь инстанс вручную — NeoForge 1.21.1, jar в папке mods.

Моды обновляются сами: в сборке есть AutoModpack — при заходе на сервер он сам докачивает обновления и предложит короткий перезапуск. Заново качать сборку не нужно.

  • В лаунчере выбери инстанс ISTHISNIXXXON / NeoForge 1.21.1 и нажми «Играть».
  • JEI — список предметов (клавиша E):

    Быстрые подсказки

    Поиск — строка «Поиск…» внизу панели

    Фильтр мода — @create, @farmersdelight

    Панель — Ctrl+O скрыть / показать

    Скрыть предмет — Ctrl+клик по иконке

  • Добавить мод отдельно: в ElyPrismLauncher открой инстанс → «Моды» → Modrinth / CurseForge, версия 1.21.1, загрузчик NeoForge.
Шаг 4

Заходи на сервер

  • IP: mc.isnix.ru (статус и онлайн)
  • Вайтлист — только на сайте: личный кабинет (ник, как обращаться, зачем на сервер). В соцсетях — только новости.
  • Заходи только после одобрения — иначе сервер не пустит.
  • После входа — хаб: зайди в obsidian-рамку слева (модовое) или справа (спокойное). Обратно — рамка у спавна в том мире.
После входа

Create — с чего начать

На сервере Create + Sable (физика, полёты): шестерни, конвейеры, поезда, карьеры. В сборке — Cyber Goggles (HUD: RPM и стресс).

  1. Андезитовый сплав → латунь
  2. Водяное колесо → шестерни
  3. Механический пресс → конвейер
  • Esc → Прогресс → ISTHISNIXXXON — цепочка достижений Create.
  • JEI: E, фильтр @create.
  • Подробнее: путеводитель на сайте.
После входа

Кланы и приваты (OPAC)

  • Меню мода: клавиша ' (апостроф) → приваты и настройки.
  • В чужой регион заходить можно, ломать и открывать сундуки — нельзя (правила, п. 3.2–3.3). Закрытый приват — преимущество владельца.
  • Клан: /openpac-parties create → invite <ник> → друг: /openpac-parties join <ник_лидера>
  • Чат клана: /opm <текст>
  • Тег клана в TAB: ' → My player config → Party name (например [Wolves]). Цвет/жирность — /clantag color green, /clantag bold on (только владелец клана).
После входа

Книга и достижения на сервере

При первом входе — книга Путеводитель ISTHISNIXXXON. Уже играл раньше или потерял книгу — напиши в чат /guidebook. В меню Esc → Прогресс — вкладка ISTHISNIXXXON (шаги по серверу) и отдельная ветка Create. Полный путеводитель на сайте →

Экономика

Торговля без валюты

  • Прямой обмен: /trade <ник> — окно обмена с игроком.
  • Рынок лотов: /sell — выставить предмет, оплата только ресурсами (без монет).
  • Чат: локально рядом; глобально — сообщение с префиксом ! в начале (см. правила на главной).
