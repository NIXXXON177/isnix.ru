# ПКМ по гробу — вещи усопшего в шалкер (админ)

Мод `isnix-player-backup` (v1.2.1+) позволяет админам забрать вещи из могилы
датапака **ly-graves** (Vanilla Tweaks Graves) правым кликом по гробу.

## Как пользоваться

1. Подойти к гробу усопшего.
2. **ПКМ** по гробу (основная рука).
3. Вещи упаковываются в шалкер(ы) и попадают в инвентарь админа.
4. **Могила исчезает** (как при обычном луте — сущности graves удаляются).

Доступ: операторы сервера (`/op`), как у `/playerbackup`.

## Имена шалкеров

| Шалкер | Название |
|--------|----------|
| 1-й | `Вещи усопшего "ник"` |
| 2-й | `Вторая часть вещей "ник"` |
| 3-й | `Третья часть вещей "ник"` |
| 4-й+ | `Часть N вещей "ник"` |

Если вещей больше 27 стопок — несколько шалкеров.

## Важно

- Работает с **реальными** вещами из могилы в мире (не копия из памяти).
- **Traveler's Backpack:** при смерти рюкзак автоматически добавляется в NBT могилы (TB без этого кладёт его только в файл). При админском ПКМ содержимое рюкзака, шалкеров и bundle раскладывается в общий список перед упаковкой.
- Обычные игроки лутают гроб как раньше (ПКМ → свои вещи).
- Команда `/graveloot` **удалена** — только ПКМ по гробу.
- Ник владельца определяется по scoreboard датапака graves (`utils.player.id`).
