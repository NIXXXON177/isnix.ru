# ISNIX GraveGuard — защита при луте могилы

Fabric **1.21.1**, серверный мод `isnix-graveguard`.

Исправляет техсбой **Vanilla Tweaks Graves v4** (ly-graves): при PvP у могилы вещи пропадают. Мод даёт **иммунитет к урону от игроков**, пока ты у **своей** могилы после смерти.

## Как работает

1. Игрок умирает → 10 мин «окно могилы» (настраивается).
2. Подошёл к своей могиле (≤ 6 блоков) или ПКМ по ней → **12 сек** без урона от игроков (таймер обновляется, пока стоишь у могилы).
3. Урон от мобов, лавы, падения **не блокируется** (по умолчанию).

Своя могила определяется по:

- тегам `graves.*` (датапack VT / ly-graves);
- нику на табличке могилы;
- UUID в тегах;
- близости к месту последней смерти.

## Команды (OP 4+)

| Команда | Описание |
|---------|----------|
| `/graveguard status` | Статус мода и (для OP) личная защита |
| `/graveguard reload` | Перечитать `config/isnix-graveguard.json` |

## Конфиг

`config/isnix-graveguard.json` — образец: [isnix-graveguard.json](config-samples/isnix-graveguard.json)

| Поле | По умолчанию | Смысл |
|------|--------------|--------|
| `protectionSeconds` | 12 | Длительность защиты (обновляется у могилы) |
| `eligibleMinutesAfterDeath` | 10 | Сколько минут после смерти действует логика |
| `nearGraveRadius` | 6.0 | Радиус поиска своей могилы |
| `protectFromPlayersOnly` | true | Только PvP; false = полный иммунитет |
| `notifyPlayer` | true | Сообщение в action bar при активации |

## Установка

1. Собрать: `cd isnix-graveguard && ./gradlew build`
2. JAR в `mods/isnix-graveguard-1.0.1.jar`
3. **Restart** сервера (не `/reload` для модов)
4. Проверка: умри → подойди к могиле → `/graveguard status`

## Связь с правилами

По **A1.2** это починка техсистемы, не `/give`. Игрокам всё равно советуем не драться у могилы — защита не бесконечная.
