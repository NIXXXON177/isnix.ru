# Полный байпас приватов OPAC для админов

Мод `isnix-opac-tab` (v1.1.0+) выдаёт полный обход защиты приватов
Open Parties and Claims через официальное API OPAC `giveFullPass(UUID)`.

## Кто получает байпас
Игрок получает полный обход, если выполнено ХОТЯ БЫ ОДНО:
- реальный оператор сервера (OP-уровень 4), ИЛИ
- есть нода LuckPerms `isnix.claims.bypass`.

Состояние применяется при входе игрока и пересинхронизируется каждые ~5 секунд,
поэтому выдача/снятие ноды подхватываются на лету (релог не нужен).

## Выдать ноду через LuckPerms
Конкретному игроку:

    lp user <ник> permission set isnix.claims.bypass true

Целой группе админов (рекомендуется):

    lp group admin permission set isnix.claims.bypass true

Снять:

    lp user <ник> permission unset isnix.claims.bypass
    lp group admin permission unset isnix.claims.bypass

## Важно
- `fullPass` в OPAC хранится В ПАМЯТИ и сбрасывается при рестарте сервера —
  мод заново применяет его на входе/в тике, отдельных действий не требуется.
- Обход — полный: ломать/ставить блоки, открывать контейнеры, взаимодействовать
  в любых чужих приватах. У обычных игроков защита продолжает работать.
