# ISNIX Market (Fabric 1.21.1)

Серверный рынок за **ресурсы** (без валюты): аналог `/ah`, оплата только предметами.

## Команды

| Команда | Действие |
|---------|----------|
| `/sell` | Открыть рынок (GUI большого сундука) |
| `/sell list` | Сразу окно «Выставить лот» |
| `/sell help` | Подсказка в чат |
| `/sell cancel <uuid>` | Снять свой лот (ID в lore или в `config/isnix-market/listings.json`) |
| `/sell buy <uuid>` | Купить весь оставшийся лот по ID |
| `/sell buy <uuid> <кол-во>` | Купить часть лота (пропорциональная оплата) |
| `/sell search <слово>` | Поиск лотов по товару (gunpowder, stick…) |
| `/sell clear` | Сбросить поиск |
| `/sell log` | Последние ваши сделки (журнал) |

## Как пользоваться

1. **`/sell`** — список лотов (до **45** на страницу), **стрелки** — страницы. **ЛКМ** по чужому лоту — окно **количества** (пропорциональная цена: 32 пороха за 32 палки → можно купить 1 за 1). **±1**, **Максимум**, **Подтвердить**.
   - **Свой лот:** **Shift+ПКМ** — снять с продажи, предмет вернётся в инвентарь.
   - Предметы из GUI **нельзя забрать** (защита от дюпа).
   - **Сундук** — «Весь рынок» / **«Мои лоты»**; **воронка** — сортировка (новые → дешевле → дороже → продавец); **лупа** — поиск (клик по предмету в инвентаре внизу) или **барьер** — сброс поиска.
   - В lore: **«За 1 шт.»** — сколько платить за одну единицу товара; предупреждение, если цена подозрительная.
   - На одного игрока — не больше **`maxListingsPerPlayer`** активных лотов (по умолчанию **10**).
2. **Изумруд «Выставить предмет»** или **`/sell list`**:
   - **товар** — клик по **инвентарю внизу** (ЛКМ — 1 шт., ПКМ — стак; списание при «Подтвердить»);
   - **цена** — клик по книге **«Ваша цена»** (каталог из конфига; **не нужно иметь** эти блоки у себя) **или** один предмет в инвентаре + кнопки **±1** (до 64);
   - с инвентаря за цену **ничего не снимается** — только задаётся, что принесёт покупатель;
   - **ПКМ** по слоту товара/цены в GUI — сброс выбора;
   - зелёная панель — подтвердить.
3. В чат при выставлении, покупке и отмене пишется **ID продажи** (`#a1b2c3d4` + полный UUID).
4. При покупке: звук у **покупателя**; у **продавца** (если онлайн) — сообщение в чат и перезвон нотного блока (настраивается в конфиге).

При **выставлении лота** в общий чат уходит сообщение (если `broadcastNewListings: true`): кто выставил, что и за сколько. Слово **купить** кликабельно — `/sell buy <uuid>`.

Оплата офлайн-продавцу копится в `pending-payouts.json` и выдаётся при входе.

## Конфиг

`config/isnix-market/isnix-market.json` (образец: [config-samples/isnix-market.json](config-samples/isnix-market.json))

| Поле | По умолчанию | Смысл |
|------|--------------|--------|
| `maxListingsPerPlayer` | 10 | Лотов на одного игрока |
| `maxListingsTotal` | 0 | Лотов на весь сервер (**0** = без лимита, только страницы в GUI) |
| `listingsExpireDays` | 14 | Автоудаление старых лотов (дни) |
| `broadcastNewListings` | true | Сообщение в чат при новом лоте |
| `banAllSpawnEggs` | true | Запрет яиц призыва |
| `bannedItemIds` | см. образец | ID предметов, которые нельзя продавать |
| `pricePresets` | см. образец | Каталог цен в GUI (полстака блоков и т.д. без предметов в инвентаре) |
| `purchaseSound` | `entity.experience_orb.pickup` | Звук у покупателя |
| `purchaseSoundVolume` | 0.7 | Громкость покупки |
| `purchaseSoundPitch` | 1.25 | Тон покупки |
| `sellerSaleJingle` | true | Три ноты (pling → chime → bell) у продавца |
| `sellerSaleSound` | `block.note_block.bell` | Один звук продавцу, если jingle выключен |
| `sellerSaleSoundVolume` | 0.75 | Громкость продажи |
| `sellerSaleSoundPitch` | 1.5 | Тон последней ноты / одиночного звука |
| `blockAbsurdRatios` | true | Блок абсурдной цены при выставлении |
| `maxUnitPriceRatio` | 64 | Макс. предметов оплаты за 1 шт. товара |
| `minUnitPriceRatio` | 0.015625 | Мин. (1/64), защита от опечатки продавца |
| `suspiciousMaxUnitPriceRatio` | 16 | Предупреждение + Shift при покупке |
| `announceOnJoin` | false | Устарело: сообщение при входе отключено в 1.6.1+ |

Данные лотов: `config/isnix-market/listings.json`  
Журнал сделок: `config/isnix-market/trades.jsonl` (обрезка: `tradeLogMaxLines`)

После правки конфига — **рестарт сервера** (reload нет).

## Установка

1. Push в `main` с изменениями в `isnix-market/` → workflow `build-isnix-market.yml` → jar в `mods/`.
2. Или `./gradlew build` в `isnix-market/` → jar в `mods/` на Play2GO.
3. **Рестарт** сервера.

Нужны **Fabric Loader**, **Fabric API**, Minecraft **1.21.1**. Клиентам мод **не нужен**.

## Права (LuckPerms)

Команда доступна всем. При необходимости:

```
/lp group default permission set minecraft.command.sell true
```

## Конфликты

Не ставьте второй мод с командой `/sell`.
