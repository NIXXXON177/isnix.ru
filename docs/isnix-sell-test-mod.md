# isnix-sell-test (клиент)

**Только для клиента.** На сервер Play2GO **не ставить** — там уже есть серверный `isnix-market`.

Помогает тестировать `/sell` на **isnix.ru**: горячие клавиши и подсказки, когда открывается GUI рынка.

## Сборка

```bash
cd isnix-sell-test
./gradlew build
```

Jar: `build/libs/isnix-sell-test-1.0.0.jar`

## Установка

1. Положи в **mods/** клиента **оба** jar:
   - `isnix-sell-test-1.0.x.jar` — горячие клавиши (клиент)
   - `isnix-market-1.2.6.jar` — сам `/sell` (в одиночке только **1.2.6**, старые версии удалить)
2. **Fabric API** обязателен.
3. Только **sell-test** без **isnix-market** → в чате будет `sell<--[ЗДЕСЬ]` — команды нет.

Готовые jar в репозитории: `downloads/isnix-sell-test-*.jar` и `isnix-market/build/libs/` или с сервера после сборки.

**Тест на реальном сервере:** достаточно **sell-test** в клиенте — на Play2GO уже стоит isnix-market. Подключись к **mc.isnix.ru**, не к «Новому миру» в одиночке без market-jar.

## Управление

Настройки → Управление → категория **ISNIX /sell (тест)**:

| Клавиша | По умолчанию | Команда |
|---------|--------------|---------|
| Открыть рынок | **B** | `/sell` |
| Выставить лот | **N** | `/sell list` |
| Справка | **H** | `/sell help` |

При открытии экранов рынка в **action bar** (над хотбаром) краткая подсказка.

## Что проверять

1. **B** — открывается GUI «Рынок ISNIX».
2. **N** — окно «Выставить лот»; товар только из инвентаря внизу.
3. Книга «Список цен» — список кнопок-цен, без выдачи предметов в инвентарь.
4. Подтверждение лота — списание товара, сообщение в чат.
5. **H** — справка от сервера в чат.

## Важно

Мод **не заменяет** серверный `isnix-market`. Без серверного мода команды `/sell` не сработают.
