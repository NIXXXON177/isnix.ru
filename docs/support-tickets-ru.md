# Личные обращения на сайте (жалобы, баги)

Игроки пишут администрации **только в личном кабинете** — не в ВК, Telegram и не в игровом чате.

## Для игроков

1. Войти: https://isnix.ru/account.html  
2. Страница **«Обращения и жалобы»**: https://isnix.ru/appeals.html  
3. Заполнить форму: тип, тема, описание, при жалобе — ник нарушителя; **прикрепить фото/видео** (до 5 файлов, до 25 МБ) и/или ссылку на запись  
4. Ждать ответа в том же диалоге и в **колокольчике** уведомлений  
5. После закрытия — вкладка **«Закрытые»**: можно **удалить** старое обращение (сообщения и файлы без восстановления). Список по **5 на страницу**.  

## Для администраторов

1. Кабинет → **Панель администрации** → вкладка **«Обращения»**  
2. Ответить в диалоге или **закрыть** обращение  
3. Вкладки **Открытые / Закрытые / Все** — пагинация по 5. Удаление только **закрытых**  
4. При личной ссоре с игроком — не вести дело (правила **A4**, передать другому админу)  

В Supabase после обновления SQL: функция `delete_support_ticket`, политика удаления файлов в `docs/supabase-support-storage.sql`.  

## Supabase (один раз)

В **SQL Editor** выполнить файл:

`docs/supabase-support-tickets.sql`

Затем для **загрузки файлов**:

`docs/supabase-support-storage.sql`

Нужны уже выполненные: `profiles`, `is_admin()`, `try_insert_user_notification`.
