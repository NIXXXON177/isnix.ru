# Short — рефералка (съёмочный лист)

**Длина:** 38–45 сек · **Формат:** вертикаль 9:16 · **Без** мема в начале и без «сидения» дольше 3 сек.

Полный каталог Shorts: [shorts-scripts-ru.md](shorts-scripts-ru.md) (Short №4).

## Название YouTube

**Основное:** `Приведи друга — префикс [Реферал] | ISTHISNIXXXON Minecraft 1.21.1`

**Короче:** `Рефералка isnix.ru — ссылка и награда 14 дней`

## Озвучка (читать с экрана, ~42 сек)

Текст без пауз на «Судную ночь» — сразу в тему.

```
Рефералка на ISTHISNIXXXON: приведи друга — получи префикс [Реферал] на 14 дней. Без доната и читов.

Заходишь на isnix.ru, в кабинете «Пригласи друга» — копируешь ссылку и кидаешь другу. Он подаёт заявку в вайтлист на сайте, твой ник подставится сам.

Когда заявку одобрят — уведомление в колокольчик. Один одобренный друг — один раз награда. Выдаём вручную, обычно за сутки.

Свой ник нельзя, альты — бан. Заявки только на isnix.ru, account.html. IP: mc.isnix.ru.
```

## Что записать заранее (чеклист)

- [ ] В профиле кабинета указан **ник Minecraft** (иначе нет ссылки).
- [ ] Запись экрана: кабинет → «Пригласи друга» → «Копировать».
- [ ] Кадр формы: поле **«Кто пригласил»** или открытие ссылки с `?ref=`.
- [ ] Кадр колокольчика с текстом про одобрение друга (не фейк, если нет события — пропусти фразу в монтаже или сделай тестовое одобрение).
- [ ] По желанию: 3–5 сек в игре с префиксом `[Реферал]` (выдать себе в LP для демо).
- [ ] 5–10 сек бега/базы для хука.
