# Миры сервера: хаб, модовое и «ванilla+» выживание

План развития ISTHISNIXXXON на **одном тарифе Play2Go** (один процесс Minecraft).

## Важно про «ваниллу»

**Настоящая ванилла без модов на клиенте** и **модовый NeoForge** — это разные клиенты. Игрок с чистым Minecraft **не зайдёт** на сервер с Create и Lost Souls-модами.

| Вариант | Клиент | Где жить |
|---------|--------|----------|
| Модовое выживание | Сборка `ISTHISNIXXXONmods.zip` (NeoForge 1.21.1) | Основной мир |
| «Vanilla+» на том же сервере | **Та же сборка**, но другой мир с более мягкими правилами (без Create-механизмов, меньше RPG) | Отдельный мир |
| Чистая ванилла | Только Java 1.21.1, без модов | **Отдельный сервер** + прокси Velocity (будущее, второй инстанс) |

## Рекомендуемая схема (сейчас → ближайшее)

```text
Игрок заходит mc.isnix.ru (NeoForge 1.21.1 + сборка с сайта)
        │
        ▼
   Мир HUB (спавн)
   — красивая площадь, NPC/таблички, правила
   — без баз и ферм (только декор)
        │
        ├── портал «Модовое выживание» ──► world_modded
        │      Create, Sable, боссы, Terralith, OPAC, Lost Souls-стек
        │
        └── портал «Спокойное выживание» ──► world_vanilla (Vanilla+)
               те же моды на клиенте, но мир проще:
               без Create-фабрик, без тяжёлых босс-модов в генерации
               (настраивается датапаками / конфигами биомов)
```

**Почему три мира, а не два:** хаб отдельно — игроки не строят у спавна, не лагают карту; модовый и «спокойный» режимы не мешают друг другу.

## Технически на NeoForge 1.21.1

1. **Сейчас:** один мир выживания + зона спавна (хаб в том же `world/` или отдельный участок).
2. **Следующий шаг:** второй мир (`world_modded/`, `world_hub/`) — мультимир через мод/плагин уровня NeoForge (Multiverse-порт, кастомные порталы, `/execute in` + datapack).
3. **Порталы на хабе:** физические врата (обсidian + мод Immersive Portals / Warp-зоны OPAC / Essential Commands warp у портала).
4. **Инвентарь:** общий или раздельный — решить до открытия второго мира (Multiverse-Inventories или отдельные инвентари по миру).

## Что не делать на одном инстансе

- Обещать «ваниллу без модов» в том же процессе, что Create — технически невозможно.
- Держать Create-фабрики у спавна — TPS и визуальный шум.

## Для сайта и анонсов

- **Сейчас играем:** модовое выживание NeoForge 1.21.1, спавн — общая точка входа.
- **В разработке:** хаб с двумя порталами (модовое / спокойное).
- **Позже (если будет второй тариф):** Velocity + Paper (ванилла) + NeoForge (моды).
